# Shopping List

A shopping list application.

# Configuration

After you clone the repository you have to add the following lines to the file `gradle.properties`:

```ini
RELEASE_KEY_ALIAS      = ???
RELEASE_KEY_PASSWORD   = ???
RELEASE_STORE_PASSWORD = ???
RELEASE_STORE_FILE     = ???
```
